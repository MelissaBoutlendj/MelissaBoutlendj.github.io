📧 Mail Spam Detection

In this project, I developed a machine learning model to classify emails as either spam or ham (legitimate email). Using the Kaggle Spam Mails Dataset, which contains email text and corresponding labels, I implemented a full pipeline including preprocessing, feature engineering, and model training.

  • Data preprocessing and text cleaning with NLTK (tokenization, stopwords removal).
  • Vectorization of text data into numerical features.
  • Exploratory data analysis with Pandas and NumPy.
  • Trained and evaluated a RandomForestClassifier with scikit-learn.

⚡ Key takeaway: This project demonstrates my ability to build practical NLP solutions using classical ML techniques for real-world classification tasks.

🏠 CROUS Checker – Intelligent Student Housing Monitoring System

Faced with the challenge of finding student housing in Lille for my Master's degree, I designed and deployed an automated solution to monitor CROUS housing availability in real time. The system instantly detects new listings and sends email notifications, giving me a major competitive edge in the highly competitive student housing market.

  • Python 3.10 – API requests and automation script.
  • GitHub Actions – Scheduled execution every 10 minutes (CI/CD, serverless).
  • SMTP (smtplib + Gmail) – Real-time email alerts.
  • Secure Environments – Encrypted credentials for safe email handling.
  • API REST JSON – Interaction with CROUS backend.

⚡ Key benefit: Immediate alerts for new housing availability, ensuring faster applications compared to other students.
